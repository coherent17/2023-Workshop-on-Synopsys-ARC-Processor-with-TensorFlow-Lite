# 2023-Workshop-on-Synopsys-ARC-Processor-with-TensorFlow-Lite

## File Structure

*   inc: header
*   src: implementation and main

## Make command
*   compile and link the object file with main
```bash
$ make
```
*   convert .elf and .map file into image
```bash
$ make flash
```
*   clean the project
```bash
$ make clean
```
